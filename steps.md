## Steps

Aloite -> Tarkistus
       -> Ei kannattajia

Tarkistus -> Vireilletulo

Vireilletulo -> Valiokuntakäsittely
             -> Hylätään
             -> Vanhentuminen

Valiokuntakäsittely -> Ensimmäinen käsittely
                    -> Vanhentuminen

Ensimmäinen käsittely -> Toinen käsittely
                      -> Hylätään
                      -> Vanhentuminen

Toinen käsittely -> Hyväksytään
                 -> Hylätään
